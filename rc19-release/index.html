AdminFaces RC19 Release

By Rafael Pestano - 17/10/2018 01:00:00 | release

We are pleased to announce that AdminFaces 1.0.0-RC19 is out!. AdminFaces is a free open source premium PrimeFaces theme and responsive template.

Below are the release highlights:

  • 14 issues were fixed in Admin template;

  • Also 14 issues were fixed in Admin theme;

  • We’ve added and released to maven central 2 new maven archetypes: Admin starter SpringBoot and Admin starter Tomcat;

  • Created AdminFaces site to gather all AdminFaces projects information;

  • Created a AdminFaces blog to make announcements and also to add posts about AdminFaces.

  • Created a AdminFaces docs to host AdminFaces documentation.

    Each release documentation is linked into the site, see here.
It is probably our latest release candidate, the next version will probably be v1.0.0!
