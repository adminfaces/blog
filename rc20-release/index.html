AdminFaces RC20 Release

By Rafael Pestano - 29/11/2018 01:00:00 | release

Table of Contents
  • Admin Template
  • Admin Theme
  • What’s next

We are glad to announce that AdminFaces 1.0.0-RC20 is out!. AdminFaces is a free open source premium PrimeFaces theme and responsive template.

This release fixes an important issue with SpringBoot integration, see details here.

Below are the release highlights:

Admin Template

Enhancements

  • Add Chinese language support (#135)

  • Create template base to share common parts of admin template (#140)

Bug fixes

  • Layout options freezes most functions (#4)

  • Restore defaults in controlsidebar is not reseting skin (#139)

  • Load template from resources folder(#137)

Infrastructure

  • Pull requests builds are failing (#136)

Admin Theme

Bug fixes

  • Spinner icon misaligned when used on datatable filters (#213)

  • Icon effect is misrendered on datatable columns (#214)

What’s next

We are working hard on Admin Addon which will provide easy setup and scaffolding (a.k.a code generation of boilerplate code) for AdminFaces.

We plan to release v1.0.0 when scaffolding is finished, that’s probably in next month or early 2019!

Always bet on Prime and Admin!
